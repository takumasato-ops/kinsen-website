Service 01

人材紹介

鳥取から、人生の転機をかたちにする。
「次の一歩」に、本気で向き合うエージェントです。

琴線の人材紹介、3つの強み

  • 地元を知り尽くした、鳥取拠点ならではのリアルな企業情報。表に出ない経営者の声・社風・実態まで踏まえてご紹介します。
  • UIターン・関西／首都圏転職にも対応する全国ネットワーク。「鳥取に帰りたい」「鳥取から都市へ挑戦したい」、どちらの想いにも寄り添います。
  • 採用後の定着・活躍まで伴走する、入社後フォロー体制。3ヶ月・6ヶ月のチェックインで、入社後のミスマッチを未然に防ぎます。

サービスの流れ

  1. キャリアカウンセリング本音と将来像を丁寧にヒアリング。求職者の「琴線」を一緒に探します。
  2. 求人提案価値観・スキルにマッチする求人だけを厳選してご紹介。
  3. 書類・面接対策通過率を高めるためのレジュメ添削・模擬面接。
  4. 面接調整・条件交渉日程調整、条件交渉まで担当エージェントが代行します。
  5. 内定〜入社決定退職交渉のサポートまで一貫して伴走します。
  6. 入社後フォロー3ヶ月・6ヶ月の定期面談で、定着と活躍を支援。

料金体系（求人企業様）

  • 求職者：完全無料（職業安定法に基づく）
  • 求人企業：成功報酬制 / 理論年収の 50％ 以内
  • 早期退職時の返金保証：1ヶ月以内 80% ／ 1〜3ヶ月 50% ／ 3〜6ヶ月 10%
  • 有料職業紹介事業 許可番号：31-ユ-300064

こんな方におすすめ

  • 鳥取で、本気のキャリアを築きたい方
  • UIターンを検討中で、信頼できるエージェントを探している方
  • 採用に課題を抱える地方企業・成長企業の経営者・人事担当者
  • 採用したあとの「定着」までを真剣に考えている企業

まずは無料相談から。お気軽にご相談ください。

Service 02

RPO（採用代行）

採用、まるごとお任せください。
戦略から実行まで、貴社の採用チームになります。

琴線のRPO、3つの強み

  • 媒体運用・スカウト・面接調整・内定者フォローまでワンストップ。採用業務の「ばらつき」を解消し、社内リソースを本来の業務に集中させます。
  • AI × プロ採用担当のハイブリッド運用で、スピードと質を両立。母集団形成からスクリーニングまで、最新ツールを駆使して効率化します。
  • スポット支援から月額契約まで、貴社の課題に合わせた柔軟なプラン。「面接調整だけお願いしたい」「採用戦略から見てほしい」、どちらにも対応可能です。

支援できる業務

  • 採用戦略立案・KPI設計
  • 求人媒体の選定・運用代行
  • 求人原稿の作成・改善
  • スカウトメール送信・候補者対応
  • 面接調整・日程管理
  • 内定者フォロー・入社準備
  • 採用レポーティング・改善提案

サービスの流れ

  1. 現状ヒアリング・課題整理採用フローのどこに詰まりがあるか、徹底分析。
  2. 採用戦略・KPI設計目標から逆算した実行プランを設計します。
  3. 実行プラン提案・契約必要な業務範囲を切り出し、最適なプランをご提案。
  4. 運用スタート媒体運用・スカウト・面接調整など、合意した範囲で実行。
  5. 週次／月次レポーティング数字と現場感の両面から改善提案を継続。

料金プラン

  • スポット支援：業務単位（5万円〜）／ピンポイントでお試しから
  • 月額固定プラン：要相談／継続的な採用支援に
  • 成果報酬型：要相談／採用成功時の支払いに振り切ったプラン

※ 貴社の採用規模・課題に応じて最適なプランをご提案します。

こんな企業様におすすめ

  • 採用担当が不足している・1人で抱えている
  • 採用戦略から見直したい
  • 媒体運用や面接調整の工数を削減したい
  • 採用ノウハウを社内に蓄積したい
  • 地方拠点の採用に苦戦している

採用課題の壁打ちから。お気軽にご相談ください。

Service 03

イベント

ひとつの「出会い」を、つぎの物語へ。
企業と人、地域と未来をつなぐ場を、私たちが創ります。

琴線のイベント、3つの強み

  • 企業×求職者、ビジネス×コミュニティ、地域×全国——あらゆる垣根を超える場づくり。「集める」のではなく「響かせる」企画設計を大切にしています。
  • 鳥取を起点に、現地ならではの体験を盛り込んだ企画力。砂丘・温泉・酒蔵・地酒など、鳥取の魅力を最大限に活かす演出が可能です。
  • 集客から運営、振り返りまで一気通貫でサポート。SNS・メルマガ・プレスリリースまで、参加者を集める仕組みもまるごと設計します。

提供できるイベント

  • 企業合同説明会・採用イベント
  • 業界別交流会・勉強会
  • 経営者・起業家コミュニティ運営
  • 地域活性化イベント・地方創生プロジェクト
  • 大規模カンファレンスの企画運営
  • 社員向けのチームビルディング合宿

サービスの流れ

  1. 目的・ターゲットのヒアリング「誰に、何を、どう届けたいか」から一緒に考えます。
  2. 企画立案・スケジューリングコンセプト設計から、当日のタイムテーブルまで設計。
  3. 会場・登壇者・コンテンツの調整琴線のネットワークで、最適なキャストを集めます。
  4. 集客・告知SNS・メルマガ・PRで、ターゲットに確実に届けます。
  5. 当日運営・進行司会・受付・撮影・記録まで現場運営をフルサポート。
  6. 振り返り・参加者フォローアンケート分析・参加者との継続的な関係構築まで支援。

こんな方におすすめ

  • 採用と広報を同時に進めたい企業
  • ブランディング強化を狙う企業
  • 地域とつながりたい・新しい挑戦をしたい方
  • 社内では人手が足りず、外部にイベント運営を任せたい方

企画の壁打ち・お見積もりはお気軽にご相談ください。
